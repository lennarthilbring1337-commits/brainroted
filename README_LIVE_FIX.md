cd generative-magazine

## How Click Handling Works Now

When you click the poster, **up to 4 handlers** can fire:

```
Click → body handler → generateCollage() ✓
  ↓
Click → document handler → generateCollage() ✓
  ↓
Click → canvas handler → generateCollage() ✓
  ↓
Click → p5.js mousePressed() → generateCollage() ✓
```

At least one will work - even if others fail!

## Troubleshooting

### Click doesn't work?
1. Open F12 → Console tab
2. Click on poster
3. Should see: `📌 Body-Click registriert auf:`
4. If not, check `DEBUGGING_LIVE.md` → "Troubleshooting" section

### Content not showing?
1. Run `python3 scanner.py` on server (generates `content/scan.json`)
2. Verify `/content/images/`, `/content/videos/` folders exist
3. Check F12 Network tab for 404 errors

### CORS errors?
1. Use relative paths: `content/images/file.jpg` ✓
2. Not: `https://domain.com/file.jpg` (causes CORS issues)
3. Contact server admin if files are on different domain
